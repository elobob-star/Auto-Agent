# Meta-Brainstorming Directory Instructions

This directory (`meta-brainstorming-docs/`) contains the foundational vision and communication history regarding the project's direction.

## Instructions for Agents
- When reading these docs, understand the "Why" behind the architecture.
- If you have new systemic epiphanies about how the project should evolve, create a new file here using the YAML frontmatter format.
- Do NOT use this directory for concrete, task-level technical findings; those belong in `knowledge_base/`.
