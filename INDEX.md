# Root Directory Index

- `AGENTS.md`: Instructions for Jules meta-agent sessions.
- `program.md`: The North Star vision for the agent.
- `harness/`: The actual agent code being tested and iterated upon.
- `knowledge_base/`: Timestamped markdown logs of findings and principles.
- `meta-brainstorming-docs/`: Vision and long-term planning documents.
- `run_tests.py`: Wrapper script to execute benchmark tests with context verbosity control.
- `results.tsv`: Historical ledger of experiment outcomes.
- `Dockerfile.base`: Base docker image for the task sandbox.
